# TOP CONTRIBUTORS GITHUB

## Requirements

+ *node version* v.8.9.1
+ *npm version* 5.5.1

## Usage
1. Install dependencies: ```npm install```
2. Start node server: ```npm start```
3. (Optional) Run tests: ```npm test```
